# Design X Develop

Engineering studio building web platforms, internal tools, and developer workflows.

We build practical software for teams that need more than off-the-shelf solutions. Our work spans marketing infrastructure, site tooling, automation, AI-assisted workflows, and maintainable web systems. [Learn More](https://designxdevelop.com)

## Focus

- Web platforms and marketing infrastructure
- Internal tools and workflow automation
- Developer tooling and AI-assisted systems
- Performance, maintainability, and technical cleanup

## Selected Projects

### [magic-shell](https://github.com/designxdevelop/magic-shell)
Type what you mean, execute what you need. An AI-powered CLI/TUI workflow for turning intent into terminal commands.

### [agent-skills](https://github.com/designxdevelop/agent-skills)
Reusable AI agent skills for coding workflows, with structured execution patterns and guardrails.

### [dxd-webflow-scraper](https://github.com/designxdevelop/dxd-webflow-scraper)
Multi-site archiver for Webflow properties, built to preserve deployable static snapshots and support backup workflows.

### [dxd-cdn](https://github.com/designxdevelop/dxd-cdn)
CDN tooling for managing and serving web assets in a more controlled, maintainable way.

### [electrobun-google-messages](https://github.com/designxdevelop/electrobun-google-messages)
Desktop wrapper for Google Messages built with Electrobun + Bun.
